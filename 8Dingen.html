8 Skills die ik wil leren

Clean Code
Clean code is de code die makkelijk te gebruiken is en makkelijk te veranderen, het is dus een simpel besturingssysteem voor programmeurs

Semantics
Semantics is de studie van de bedoeling van linguistische expressies

Animaties
Animaties zijn bij elkaar gevormde pixels die er toe doen laten lijken dat het beweegt

Performance
Hoe communiceer je met de klant en hoe spreek je het af

Responsive design
Responsive webdesign is een benadering van webdesign waarbij de web developer streeft naar een optimale webervaring

Search engine optimization
Het ervoor zorgen, dat de zoekopdrachten zo accuraat mogelijk zijn

Accessibility
De toegankelijkheid van een website of webpagina voor gebruikers met een handicap.

Web Technologies
Web technology is het samenstellen en gebruiken van mechanismes die het mogelijk maken voor verschillende computers en apparaten dingen te delen
